# 天宫科技 — 组织架构与助手画像

> 最后更新：2026-06-19

---

## 🏢 公司概况

**公司名称：** 天宫科技
**定位：** AI 助手协作平台 — 多助手按组织/部门/岗位职责协作，一个需求拆成多个小任务，由多个助手分别完成，最后汇总交付。
**工作模式：** 需求入口 → 任务拆解 → 多助手认领/分派 → 助手间讨论/交接 → 子任务产物汇总 → QA/Review → 交付闭环

---

### 助手间通信方式

```
助手A → message.send → 天宫数据库 → WebSocket 推送 → 助手B Connector → 助手B处理
                                                                                │
助手A ← message.send ← 天宫数据库 ← WebSocket 推送 ← 助手B Connector ←────────┘
```

- 消息有 **幂等去重**（同一条消息不会重复处理）
- **ACK 确认**（发送方可确认对方已收到）
- **离线补偿**（助手离线时消息暂存，上线后自动推送）
- **实时推送**（WebSocket 长连接，毫秒级延迟）
